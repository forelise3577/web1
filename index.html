HTML

HTML은 Hyper Text Markup Language 약어로
HyperText(웹 페이지에서 다른 페이지로 이동할 수 있도록 하는 것) 기능을 가진 문서를 만드는 code입니다.
다시 말해, 구조를 설계할 때 사용되는 언어로 hyper link 시스템을 가지고 있으며,
흔히 말하는 웹 페이지를 위한 마크업 언어라고 할 수 있습니다.

개인적으로 html 태그를 공부하고 있는 페이지 입니다.
개연성 없는 태그가 난무할수 있습니다.
화이팅!!!!


아래는 html tag 네임과 예제 입니다.


  1. <a> : Anchor

    2. 예제 : W3Schools.com에 대한 링크를 만듭니다.


    결과 : Visit W3Schools.com!

    <a> 태그는 한 페이지에서 다른 링크를 사용하는 하이퍼 링크를 정의합니다.
    <a> 요소의 가장 중요한 속성은 링크의 목적지를 나타내는 href(hypertext reference 약자) 속성 입니다.

    기본적으로 링크는 모든 브라우저에서 다음과 같이 나타납니다.

    • 방문하지 않은 링크는 밑줄과 파란색으로 표시됩니다.
    • 방문한 링크는 밑줄과 자주색으로 표시됩니다.
    • 활성 링크는 밑줄과 빨간색으로 표시됩니다.
